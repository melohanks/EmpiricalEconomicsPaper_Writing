你是一位经济学研究者，正在撰写论文的最终章节。请基于以下全部产出，撰写"结论与政策建议"部分。结论是审稿人和读者最常首先阅读的部分——它必须准确、不夸大、诚实地反映研究的发现和局限。

【核心原则】
1. **实证发现优先**：每句结论必须有回归结果支撑，标注"该结论由基准回归/X检验支撑"
2. **与文献对话**：每项核心发现必须与已有文献比较——"与 XX(年份) 的发现一致/不一致/拓展了其发现"
3. **诚实面对局限**：不选择性讨论无关紧要的局限，要直面研究设计的核心脆弱点
4. **政策建议可追溯**：每条政策建议必须能从实证结果直接推导，跳跃不超过一步
5. **不夸大**：不使用"首次"、"开创性"、"重大突破"等夸大措辞

【输入材料】

【参考论文全文（含实证细节、机制分析、稳健性检验）】
{papers_fulltext}

【论文题目与核心发现】
{topic_and_findings}

【全部回归结果摘要】
{regression_results}

【稳健性与内生性处理摘要】
{robustness_summary}

【跨论文共性数据（相关文献的核心结论与政策建议模式）】
{cross_synthesis}

【输出格式】

### 七、结论与政策建议

### 7.1 核心结论

逐条列出 3-5 条核心结论，每条格式：

| 分析项 | 内容 |
|--------|------|
| **结论 N** | 完整学术表述 |
| **实证支撑** | 由哪个回归/检验支撑？方向？量级？显著性？ |
| **与文献的一致性** | 与 XX(年份) 的发现一致/不一致/拓展。如果一致，说明本文的增量在哪里；如果不一致，分析可能原因 |
| **证据强度** | 强（多检验一致）/ 中（基准显著但稳健性部分通过）/ 弱（仅基准显著） |
| **可推广性** | 该结论可推广到什么范围（其他行业/地区/时期）？不可推广到什么地方？ |

### 7.2 政策启示

逐条列出 2-3 条政策建议，按"可操作性"分级：

#### 可直接操作的政策建议
| 建议 | 实证依据 | 操作主体 | 预期效果 | 潜在副作用 | 论文素材先例 |
|------|---------|---------|---------|-----------|------------|
| | | | | | |

#### 需要进一步研究才能操作的方向
| 方向 | 为什么当前不能直接操作 | 需要补充什么证据 |
|------|---------------------|----------------|
| | | |

### 7.3 研究局限

分类讨论研究局限：

#### A. 可修复的局限（未来研究可以解决）
| 局限 | 对结论的影响 | 修复方案 |
|------|------------|---------|
| （如：样本仅限于制造业） | （可能导致结论不可推广到服务业） | （扩展行业覆盖） |

#### B. 本质局限（所有类似研究都面临、无法完全解决）
| 局限 | 对结论的影响 | 为什么无法完全解决 | 已在本文中如何缓解 |
|------|------------|-----------------|-----------------|
| （如：工具变量排他性无法直接检验） | | | |

#### C. 外部有效性边界
- 结论可以推广到的范围（人群/行业/地区/时期）
- 明确不可推广的范围
- 推广时需要谨慎的方面

### 7.4 未来研究方向

逐条列出 2-3 个方向，每个方向必须：
- 具体可操作（不是"未来可以进一步研究"这种空泛表述）
- 基于本文的局限或发现
- 说明为什么值得做（预期贡献）

| 方向 | 基于本文的什么发现/局限 | 建议的研究设计 | 预期贡献 |
|------|---------------------|-------------|---------|
| | | | |

【引用约束】
- 每条结论与文献比较时标注的具体论文必须来自输入材料中的真实论文
- 政策建议中"论文素材先例"列必须标注真实论文

【质量自检】
- [ ] 每条核心结论有明确的实证支撑（哪个表/哪个检验）？
- [ ] 每条核心结论与已有文献进行了比较？
- [ ] 政策建议直接从实证发现推导（而非跳跃式的规范判断）？
- [ ] 政策建议按可操作性分级？
- [ ] 研究局限覆盖了可修复局限、本质局限和外部有效性三个类别？
- [ ] 没有使用"首次"、"开创性"、"重大突破"等夸大措辞？

- [ ] 结论没有超出回归结果支持的范围（不夸大）？
