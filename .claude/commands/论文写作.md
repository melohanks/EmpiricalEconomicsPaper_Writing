请启动论文写作交互流程。每一步都需要用户确认后再继续。

## 当前状态检查

1. 首先运行 `agent.get_projects_summary()` 检查已归档 + 当前工作区的全部论文分析来源
2. 展示给用户：N 个论文来源、各来源的分析覆盖情况
3. **关键**：选题/假设/模型/变量步骤会**自动聚合所有已归档论文的分析报告**（`workspace/projects/*/` + 当前 `workspace/`），而非只读当前一篇
4. 如果没有任何分析来源（0 个），先引导用户运行 `/文献综述` 和 `/实证分析`

## 交互流程

### Step 1: 选题

运行 LLM 提取所有分析报告中的 Research Gaps，生成3-5个候选题目。

```bash
cd "d:\全流程论文写作" && export ANTHROPIC_API_KEY="$ANTHROPIC_API_KEY" && export ANTHROPIC_BASE_URL="$ANTHROPIC_BASE_URL" && export LLM_BACKEND="anthropic" && export ANTHROPIC_MODEL="$ANTHROPIC_MODEL" && export PYTHONIOENCODING="utf-8" && python scripts/run_write.py --backend anthropic --model $ANTHROPIC_MODEL --step topic
```

展示候选题目列表，让用户选择或修改。

### Step 2: 提出假设

基于用户确认的题目，生成研究假设 H1/H2/H3。

```bash
cd "d:\全流程论文写作" && export ANTHROPIC_API_KEY="$ANTHROPIC_API_KEY" && export ANTHROPIC_BASE_URL="$ANTHROPIC_BASE_URL" && export LLM_BACKEND="anthropic" && export ANTHROPIC_MODEL="$ANTHROPIC_MODEL" && export PYTHONIOENCODING="utf-8" && python scripts/run_write.py --backend anthropic --step hypothesis --input "用户确认的题目和变量信息"
```

### Step 3: 模型选择

统计论文素材中的模型使用频率，推荐基准模型。

```bash
cd "d:\全流程论文写作" && export ANTHROPIC_API_KEY="$ANTHROPIC_API_KEY" && export ANTHROPIC_BASE_URL="$ANTHROPIC_BASE_URL" && export LLM_BACKEND="anthropic" && export ANTHROPIC_MODEL="$ANTHROPIC_MODEL" && export PYTHONIOENCODING="utf-8" && python scripts/run_write.py --backend anthropic --step model --input "题目+假设摘要"
```

### Step 4: 变量选取

推荐变量体系（Y / X / 中介变量 / 控制变量 / IV），引用论文素材中的测度惯例。

```bash
cd "d:\全流程论文写作" && export ANTHROPIC_API_KEY="$ANTHROPIC_API_KEY" && export ANTHROPIC_BASE_URL="$ANTHROPIC_BASE_URL" && export LLM_BACKEND="anthropic" && export ANTHROPIC_MODEL="$ANTHROPIC_MODEL" && export PYTHONIOENCODING="utf-8" && python scripts/run_write.py --backend anthropic --step variables --input "题目+模型摘要"
```

### Step 5: 实证回归（需要数据后执行）

蓝图完成后，可运行 `/实证回归` 进行 Stata 实证分析：
- 读取蓝图中的模型/变量规格
- 通过 Stata MCP 依次运行：基准回归 → 核心识别 → 机制检验 → 异质性 → 稳健性
- 结果回收写入 `workspace/regression/empirical_results.md`

### Step 6: 撰写全文

汇总全部分析材料 → 生成完整论文初稿 → `workspace/writing/full_paper.md`
