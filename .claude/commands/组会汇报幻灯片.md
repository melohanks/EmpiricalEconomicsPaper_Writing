请生成组会汇报幻灯片和演讲稿。

## 前提检查

1. 确认 `workspace/literature_review.md` 存在
2. 确认 `workspace/analysis/` 目录下有11个维度的分析文件
3. 确认 `workspace/papers/metadata.json` 有论文数据
4. 如果任一缺失，提示用户先运行 `/文献综述`

## 生成 PPT（首选 ppt-master）

使用 **ppt-master skill** 生成学术报告 PPTX：

1. 准备输入材料：将文献综述核心内容 + 11维分析关键发现整合为 Markdown 文档
2. 调用 `Skill(skill="ppt-master", args="基于以下文献综述和分析报告生成学术组会PPT...")`
3. ppt-master 会将内容转化为多页 SVG 页面并导出为 PPTX

### 提供给 ppt-master 的内容结构

```
# 组会汇报：<论文标题>
## 研究背景与问题
## 文献缺口与本文贡献
## 理论框架与研究假设
## 识别策略与方法论
## 数据与变量体系
## 基准结果
## 机制检验
## 异质性分析
## 稳健性与内生性
## 结论与政策建议
## 研究局限与展望
```

每个章节包含3-5个要点，附带关键数据/表格。

## 备用方案（HTML）

如果 ppt-master 不可用，回退到 Python 脚本生成 HTML 演示：
```bash
cd "d:\全流程论文写作" && python -c "
import json, os, sys
sys.path.insert(0, '.')
from skills.presentation_generator import PresentationGenerator

with open('workspace/literature_review.md', 'r', encoding='utf-8') as f: review = f.read()
with open('workspace/papers/metadata.json', 'r', encoding='utf-8') as f: papers = json.load(f)

results = []
for fn in sorted(os.listdir('workspace/analysis')):
    if fn.endswith('_analysis.json') and not fn.startswith('_'):
        with open(os.path.join('workspace/analysis', fn), 'r', encoding='utf-8') as f:
            d = json.load(f)
            results.append({'section_key': d.get('section_key',''), 'title': d.get('section_title',''), 'json': d, 'success': True})

gen = PresentationGenerator()
r = gen.execute('generate_all', review_text=review, section_results=results, papers=papers, backend='anthropic', model='deepseek-v4-pro')
h = r.get('html', {})
s = r.get('speech', {})
print(f'幻灯片: {h.get(\"slide_count\",\"?\")} 页 -> {h.get(\"html_path\",\"失败\")}')
print(f'演讲稿: {\"成功\" if s.get(\"success\") else \"失败\"} -> {s.get(\"speech_path\",\"\")}')
"
```
