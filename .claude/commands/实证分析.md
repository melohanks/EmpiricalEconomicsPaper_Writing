请执行实证方法论深度分析。

## 1. 前提检查

确认 `workspace/papers/metadata.json` 中有论文元数据。如果为空，先引导用户使用 `/论文获取` 或 `/文献综述`。

## 2. 运行实证分析

```bash
cd "d:\全流程论文写作" && python scripts/run_empirical.py --backend anthropic --model $ANTHROPIC_MODEL
```

如果想进行多篇论文的横向比较，加 `--compare` 参数：
```bash
cd "d:\全流程论文写作" && python scripts/run_empirical.py --backend anthropic --model $ANTHROPIC_MODEL --compare
```

## 3. humanizer-zh 去AI味

对 `workspace/empirical/` 下的所有 `*_empirical.md` 文件执行去AI味：

- 读取每个文件 → 调用 `Skill(skill="humanizer-zh", args="<文件内容>")`
- 将去AI味后的文本写回对应文件

**注意**：保留学术术语、表格结构、变量名、统计量不变。

## 4. 报告结果

向用户展示：
- 四维度分析覆盖情况（研究假设、实证方法、变量体系、实证结果与逻辑链条）
- 输出文件路径（`workspace/empirical/`）
- 关键方法论发现摘要
- humanizer-zh 处理状态
- 如有横向比较，展示方法对比矩阵与创新空间推断
