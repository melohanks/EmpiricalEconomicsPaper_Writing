请从 Zotero 本地库获取论文。

## 解析用户输入

用户可能在 `/论文获取` 后提供：
- **关键词**：如 `耐心资本` `供应链韧性` → 按标题搜索
- **全部**：如 `--all` `全部` → 导出所有期刊论文
- **指定key**：如 `KMJDXCRI` → 导出单篇

## 执行步骤

### 1. 列出 Zotero 库中的论文（可选）

```bash
cd "d:\全流程论文写作" && python scripts/zotero_extract.py --list --limit 20
```

展示给用户，确认要导入哪些。

### 2. 按关键词搜索并导出

```bash
cd "d:\全流程论文写作" && python scripts/zotero_extract.py --search "用户输入的关键词" --output workspace/papers/metadata.json --limit 10
```

或导出全部：

```bash
cd "d:\全流程论文写作" && python scripts/zotero_extract.py --all --output workspace/papers/metadata.json --limit 50
```

### 3. 检查结果

读取 `workspace/papers/metadata.json` 确认：
- 论文标题、作者、期刊、摘要、关键词、DOI 均完整
- 无重复记录（脚本自动按 zotero_key 去重）

### 4. 从 Python 直接读取展示结果

由于终端编码限制，使用 Python 直接打印 UTF-8 内容：
```bash
cd "d:\全流程论文写作" && python -c "
import json
with open('workspace/papers/metadata.json', 'r', encoding='utf-8') as f:
    papers = json.load(f)
print(f'共 {len(papers)} 篇论文:')
for p in papers:
    print(f'  [{p.get(\"zotero_key\",\"?\")[:8]}] {p[\"title\"][:60]}')
    print(f'    作者: {p[\"authors\"]}  |  {p[\"source\"]} ({p[\"pub_date\"]})')
    kw = p.get('keywords', [])
    if kw: print(f'    关键词: {\", \".join(kw[:5])}')
    if p.get('attachments'): print(f'    PDF: {p[\"attachments\"][0][\"path\"][:60]}')
"
```

### 5. 报告结果

向用户展示：
- 从 Zotero 提取的论文数量
- 每篇的标题、作者、关键词
- PDF 附件情况
- 提示：下一步可运行 `/文献综述`

## Zotero 数据库

位置：`C:\Users\User\Zotero\zotero.sqlite`（自动读取，无需配置）
脚本：`scripts/zotero_extract.py`（直接读取 SQLite + 自动去重）
