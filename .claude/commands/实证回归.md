## 5. 基准回归（H1）

创建 `workspace/regression/do/03_baseline.do`：

```stata
clear all
capture log close
log using "workspace/regression/log/03_baseline.log", text replace
use "workspace/data/analysis_sample.dta", clear

xtset firm_id year

* (1) OLS
reg Y X controls, robust
estimates store m1

* (2) 面板FE
xtreg Y X controls, fe cluster(firm_id)
estimates store m2

* (3) 双向FE
reghdfe Y X controls, absorb(firm_id year) cluster(firm_id)
estimates store m3

* 输出表格
estimates table m1 m2 m3, star(0.10 0.05 0.01) stats(N r2)

* 导出 CSV
esttab m1 m2 m3 using "workspace/regression/tables/baseline.csv", ///
    replace star(* 0.10 ** 0.05 *** 0.01) ///
    stats(N r2 r2_a, fmt(%9.0f %9.3f %9.3f))

log close
```

**提取关键信息**：核心系数 + 标准误 + 显著性 + R² + N → 判定 H1
